
### Layout 布局

通过基础的 24 分栏，迅速简便地创建布局。

::: demo 这是一个简单的 `Button`, 下面中演示地代码将会自动转化成一个按钮
```html
<template>
  <button @click="hello">对这个世界多一点善意</button>
</template>

<script>
  export default {
    methods: {
      hello() {
        alert('Hello World!');
      }
    }
  }
</script>
```
:::
